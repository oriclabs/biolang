Getting Started

Loading Files

Drag & drop any supported file onto the viewer, use the file picker, or paste text with Ctrl+V. Pasted bioinformatics data (starting with >, @, ##, etc.) is auto-detected and opened as a new tab.

URL loading: Append ?url=https://example.com/data.vcf to load a remote file. If the remote server lacks CORS headers, BioPeek automatically retries through a CORS proxy fallback.

GZ decompression: .gz files are transparently decompressed using the browser's native DecompressionStream API. Drop a .vcf.gz, .fastq.gz, or any gzipped file and it is decompressed on the fly.

Smart paste: Pasted content is auto-detected — raw sequences become FASTA, tab-delimited genomic rows become BED, comma-separated data becomes CSV. No manual format selection needed.

Binary files: BAM, BCF, and CRAM are detected and cannot be viewed directly. Convert to text formats first.

Table View

Sorting, Search, Column & Cell Menus

Sort: Click a column header to cycle ascending/descending/unsorted. Arrow indicators show sort direction. Hold Shift+click for multi-column sort with priority numbers.

Ascending
Descending

Search: Press / to focus the search box. Supports regex patterns. Capture groups show extracted matches in a floating panel (e.g., AF=([0-9.]+)).

Column header tooltips: Hover numeric headers for min/max/mean/median + inline sparkline histogram.

Inline column filters: Click the filter icon in any header to show a checklist of unique values. Multiple column filters combine with AND logic. Paste a list of values to bulk-filter.

Column context menu (right-click header): Pin, Copy Name, Copy Column, Unique Values, Change Type, Conditional Format, Numeric Format, Filter.

Cell context menu (right-click cell): Copy Cell, Copy Row, Reverse Complement, BLAST Sequence, Open in dbSNP/GeneCards/Ensembl.

Column resize: Drag the right edge of any header. Double-click the handle to auto-fit width.

Column reorder: Drag headers to rearrange. A blue indicator shows placement.

Double-click header: Opens a histogram sidebar with distribution chart and column stats.

Features

Bookmarks & Row Detail

Bookmarks: Click Bookmarks to reveal a star column. Click stars to flag rows. Click the filled star in the header to export only bookmarked rows.

Row detail: Enable the Detail toggle, then click any row to open a detail panel. VCF INFO fields are auto-expanded into labeled key=value pairs.

Selection stats: Ctrl+click rows to multi-select. The footer shows count, sum, average, min, max for the selection.

Tabs & Diff

Tab colors: Tabs are color-coded by format: VCF=cyan, FASTA=green, FASTQ=violet, BED=amber, GFF=pink, SAM=orange, CSV/TSV=slate. Row count badges update with filter state.

Tab actions: Double-click to rename. Right-click to duplicate. Hover for tooltip with size, rows, format, and quality score.

Diff: When 2+ same-format files are open, click Diff to compare. Rows are matched by natural keys (VCF: CHROM:POS:REF>ALT, BED: CHROM:START-END, FASTA/FASTQ: ID). Shows shared, only-in-A, only-in-B counts.

Row diff: Select exactly 2 rows with Ctrl+click, then Diff to see a side-by-side comparison with differences highlighted.

Editing, Find & Replace, Undo

Cell editing: Double-click any cell to edit in place. Enter to confirm, Esc to cancel. Edited cells are highlighted.

Find & Replace: Click Replace to expand the replace bar. Supports single or Replace All on text columns.

Undo: Ctrl+Z reverts edits, replacements, column deletes, and sort resets. Multiple levels supported.

In-memory only: Edits do not modify the original file. Export to save changes.
Conditional Formatting & Column Types

Conditional formatting: Right-click a header, select Conditional Format. Set operator (equals, greater/less than, contains, between), threshold, and highlight color. Multiple rules stack.

Column type override: Right-click header → Change Type. Options: text, numeric, sequence. Changing to numeric enables stats; sequence enables GC%, reverse complement, BLAST.

Numeric formatting: Right-click numeric header → Format. Choose fixed decimals (0-6), scientific notation, or percentage.

GC% auto-offer: Columns with DNA/RNA sequences show a GC% badge in the header. Click to add a computed GC% column.

Export, Screenshots, Sharing

Export: CSV, TSV, BED, VCF, HTML (self-contained). Format conversions: VCF→BED, GFF→BED, FASTA→reverse complement.

Screenshot: Click Screenshot to export current view as PNG with headers, up to 30 rows, ATCG coloring, and current theme.

Share links: Click Share for files <100 KB. Data is base64-encoded in the URL. Only works on public deployments (not extension or localhost).

Export HTML: Standalone HTML file with all styling included, shareable via email.

Sequence Tools & External Links

Sequence selection popup: Select DNA/RNA text in a cell to see reverse complement, GC content, and 3-frame codon translation with copy buttons.

BLAST: Right-click any sequence cell (≥10 bp) to open NCBI BLAST with the sequence pre-filled.

GZ Decompression

BioPeek transparently decompresses .gz and .bgz files using the browser's native DecompressionStream API. Drop a .vcf.gz, .fastq.gz, .fa.gz, .bed.gz, or any gzipped bioinformatics file and it is decompressed on the fly without any server.

This works across all entry points: drag-and-drop onto the viewer, file picker, extension popup, context menu ("Open in BioPeek"), and URL loading. Files up to 7 MB can be opened from the extension popup; larger gz files open the viewer directly for drag-and-drop.

Browser support: Requires Chrome 80+, Firefox 113+, or Safari 16.4+. Older browsers will show a decompression error with a gunzip command hint.
Chunked Preview for Large Files

Files larger than 5 MB are loaded using chunked preview. The first chunk is parsed and displayed immediately while the rest of the file continues loading in the background. You can start exploring the table without waiting for the full file.

Genomic Coordinate Navigation

Press Ctrl+Shift+G to search by genomic coordinate. Type chr1:12345 or chr1:10000-20000 to filter the table to rows overlapping that region. Works with VCF, BED, GFF, and SAM files.

FASTQ QC Panel

FASTQ files display a dedicated QC panel in the Stats view with read statistics, Q30% metric, per-base quality histogram, per-position quality chart, and GC content distribution.

FASTQ Quality Heatmap Bars

In the table view, FASTQ quality strings are rendered as colored heatmap bars: green for high quality (≥ Q30), amber for medium (Q20–Q29), and red for low (< Q20).

VCF Variant Density Chart

VCF files show a per-chromosome variant density chart in the Stats view. Click a chromosome bar to filter the table to that chromosome.

VCF FILTER Pie Chart

VCF Stats view includes a donut chart showing the distribution of FILTER values (PASS vs. filter flags). Click a segment to filter the table to that FILTER value.

Strand Indicators

BED and GFF files display visual strand indicators: plus strand (+) as a right-pointing green arrow, minus strand (-) as a left-pointing rose arrow.

Protein Sequence Support

BioPeek auto-detects protein FASTA files (.faa or by content analysis). Protein sequences are colored by physicochemical property using the Zappo scheme:

PropertyResiduesColor
HydrophobicA, I, L, M, VAmber
AromaticF, W, YPurple
PolarS, T, N, QGreen
Positive chargeK, R, HBlue
Negative chargeD, ERed
SpecialC, G, PCyan

Protein files show residue count instead of bp, display amino acid composition statistics (top residues), and skip the GC% column offer since it's irrelevant for protein sequences.

Codon Coloring for DNA FASTA

DNA FASTA sequences display codon-level coloring with alternating tint bands per triplet. Nucleotides use standard coloring: A green, T red, C blue, G amber, N grey.

Sequence Motif Search

Search for DNA/RNA motifs using IUPAC ambiguity codes (e.g., TATAWR, CANNTG) or regex patterns. Matching positions are highlighted inline within sequence cells.

Copy as BioLang / FASTA / BED

Right-click the table view to "Copy as BioLang" — generates pipe code reproducing your current filters and sort. Right-click cells for "Copy as FASTA" or "Copy as BED" for format-aware clipboard output.

PostMessage File Receive

BioPeek listens for postMessage events from other browser contexts. The BioGist extension uses this to send detected file links directly to BioPeek for visualization.

Inline Docs Hover Tooltips

In the BioLang console, hovering over builtin function names shows inline documentation tooltips with the function signature, description, and usage example.

Large File Handling

File Size Thresholds & Streaming Stats
File SizeBehavior
< 5 MBFull content cached in IndexedDB for session restore. Chunked preview loads first chunk immediately for files > 5 MB.
< 10 MBFull text in memory for raw view and console
< 50 MBFull text parsed; raw view shows all content
> 50 MBFirst 50 MB parsed. "Compute full stats" button streams the entire file.
Table viewDOM limited to 5,000 rows for scroll performance; use search/filter to narrow

Streaming stats for large files process data in 8 MB chunks using Welford's online algorithm (exact mean/variance) and reservoir sampling (k=1000 for approximate median/percentiles). A progress bar shows streaming progress with incremental stat updates.

Chrome Extension

Extension-Specific Features

The Chrome extension provides BioPeek directly in your browser with additional integration:

  • Context menu: "Open in BioPeek" — Right-click any link to a bioinformatics file (.vcf, .fasta, .bed, etc.) to open it directly in BioPeek
  • Context menu: "Analyze selection in BioPeek" — Select sequence text on any webpage, right-click to analyze it in BioPeek
  • Download interception — Automatically offers to open downloaded bio files in BioPeek
  • Popup quick-open — Drop files or paste URLs in the extension popup for instant viewing
Extension limitations: Share links and PWA features (file handling, install prompt, session restore) are not available in the extension due to chrome-extension:// URLs.

v1.3.0 — New Features

NCBI Sequence Fetch

Fetch FASTA sequences directly from NCBI by accession number. Enter a GenBank, RefSeq, or UniProt accession (e.g., NM_007294, NC_000017, NP_009225, P38398) in the input on the drop zone or click the NCBI toolbar button.

Batch fetch: Enter multiple accessions separated by commas or spaces. Each is fetched sequentially with rate limiting (350ms delay). Results are combined into a multi-sequence FASTA file.

Tries NCBI nucleotide database first, falls back to protein database, then UniProt. All requests have a 15-second timeout.

Sequence Color Toggle

Click the Color toolbar button to toggle nucleotide/amino acid coloring on or off. When on, the button shows an amber highlight. When off, sequences display as plain monospace text. Setting is persisted across sessions.

GC% Sliding Window Plot

In the Stats view for FASTA files, a line chart shows GC% across a 100bp sliding window for the first sequence. The Y-axis ranges from 0-100% with gridlines at 25% intervals. A dashed amber line marks the 50% reference. Approximately 200 sample points are plotted.

K-mer Frequency Table

In the Stats view for FASTA files, a table shows the top 20 most frequent 4-mers with count, frequency percentage, and an inline bar chart. Sampled from the first 5 sequences. Only ACGT k-mers are counted (N-containing k-mers are skipped).

Chromosome Multi-Select

In VCF/BED/GFF/SAM files, clicking chromosome tags in the strip now supports multi-select. Click chr1, then click chr2 to see both. Active tags show an accent highlight. Click an active tag again to deselect it. Deselecting all clears the filter.

Large File Streaming Mode

Files larger than 10 MB activate streaming mode automatically. BioPeek scans the file to count records (without loading the full content), then reads and parses the first 5 MB as a preview. This keeps memory usage low and the UI responsive.

A cyan banner shows: "Showing 1,200 of 500,000 total records (preview mode)." Click Load All Records to read and parse the entire file when you need full sort/filter capabilities.

Supported formats: FASTA, FASTQ, VCF, BED, GFF, CSV, TSV. Files under 10 MB use the existing full-parse path with no changes.

Sort & Filter Column Icons

Column headers now show visible indicators: a sort arrow (⇅) and a filter funnel icon. Click the column name to sort. Click the funnel icon to open the filter dropdown. The sort arrow turns accent-colored when active. The funnel turns green when a filter is applied. Column resize handles show as a visible border on the right edge of each header.

Tooltip: Hover any column header to see: "Click: sort | funnel: filter | Double-click: column details"
