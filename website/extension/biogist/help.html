Getting Started

Quick Start
  1. Click the BioGist icon in your toolbar to open the sidebar
  2. Visit any scientific paper — PubMed, bioRxiv, Nature, Cell, Science, etc.
  3. Click "Scan Page" — BioGist detects genes, variants, accessions, and species on the page
  4. Click any entity for instant details from NCBI, UniProt, and gnomAD
On bio-relevant pages (PubMed, bioRxiv, etc.), scanning happens automatically when you open the sidebar.

Features

Entity Highlighting

After scanning, BioGist highlights detected entities directly on the page. Genes appear in purple, variants in cyan, and accessions in amber. Click any highlighted term to jump to its details in the sidebar.

Gene Details

Click a detected gene to see:

  • NCBI Gene summary — official full name, description, and gene ID
  • UniProt protein function — protein function, subcellular localization, and UniProt accession
Variant Details

Click a detected variant (rsID, HGVS, ClinVar, or COSMIC) to see:

  • dbSNP info — variant class, alleles, and genomic position
  • gnomAD allele frequency — population frequency across global cohorts
  • ClinVar significance — clinical interpretation (pathogenic, benign, VUS, etc.)
Dataset Details

Click a GEO or SRA accession to see:

  • GEO metadata — title, summary, platform, sample count, organism
  • SRA metadata — experiment type, library strategy, read count
Copy All IDs

One click copies all detected accessions as a newline-separated list, ready to paste into a spreadsheet, script, or search tool.

Open in BLViewer

When BioGist detects links to bioinformatics files (FASTQ, VCF, BED, etc.) on the page, click "Open in BLViewer" to send them directly to BLViewer for visualization and analysis.

SRA Download Command

For detected SRR accessions, BioGist generates a ready-to-use fasterq-dump command:

fasterq-dump --split-3 SRR1234567 -O ./output

Click to copy the command to your clipboard.

Search & Filter

Use the search box at the top of the sidebar to filter detected entities by keyword. Matches are filtered across all entity types in real time.

Right-Click Lookup

Select any text on a webpage, right-click, and choose "Look up in BioGist" from the context menu. BioGist will attempt to identify it as a gene, variant, or accession and show details in the sidebar.

24-Hour Caching

API results from NCBI, UniProt, and gnomAD are cached locally in Chrome storage for 24 hours. Repeat lookups for the same entities load instantly without additional API calls.

Tips

Getting the Best Results
  • Best supported sites: PubMed, bioRxiv, Nature, Cell, Science, PLOS, and GitHub README pages
  • Auto-scan: Bio-relevant pages are detected automatically and scanned when the sidebar opens
  • Manual scan: For non-bio pages or pages not auto-detected, click "Scan Page" manually
  • Right-click lookup: When auto-scan misses something, select the text and use the right-click context menu
  • Gene accuracy: The curated gene list avoids false positives from common English words. If a gene is not detected, use right-click lookup
  • Multiple papers: Each tab maintains its own scan results — switch between tabs to compare entities across papers

Privacy

Data Handling
  • Local scanning — all page scanning and entity detection runs entirely in your browser
  • Direct API calls — lookups go directly to NCBI, UniProt, and myvariant.info (no proxy server)
  • No data uploaded — no data is sent to any Oric Labs server, ever
  • No analytics — no telemetry, tracking, or usage analytics of any kind
  • Local cache only — API results are cached in local Chrome storage and never leave your machine

Troubleshooting

Common Issues
ProblemSolution
No entities detected The page may not contain recognized gene symbols or accessions. Try selecting specific text and right-clicking → "Look up in BioGist".
API errors or timeouts NCBI has rate limits (3 requests/sec without an API key). Wait a moment and try again. Cached results are not affected.
Sidebar not opening Ensure the extension is enabled at chrome://extensions. Try refreshing the page, then clicking the icon again.
Highlights not visible Some sites use CSS that conflicts with highlighting. Entity results are still available in the sidebar even if page highlights do not appear.
Missing gene symbol The curated list covers 334 common human genes. For unlisted genes, select the name and use right-click → "Look up in BioGist".
