# Day 10: Tables --- The Bioinformatician's Workbench

| | |
|---|---|
| **Difficulty** | Intermediate |
| **Biology knowledge** | Basic (gene names, chromosomes, expression data) |
| **Coding knowledge** | Intermediate (pipes, closures, records) |
| **Time** | ~3 hours |
| **Prerequisites** | Days 1-9 completed, BioLang installed (see Appendix A) |
| **Data needed** | Generated by `init.bl` (CSV files) |
| **Requirements** | None (offline) |

## What You'll Learn

- How to create, filter, mutate, sort, group, join, and reshape tables
- The split-apply-combine pattern (group_by + summarize)
- Six join types for connecting related datasets
- Pivoting between wide and long formats
- Window functions for running totals and ranks
- Building complete analysis pipelines with pipes

## Files

| File | Description |
|------|-------------|
| `init.bl` | Generates sample CSV files in `data/` |
| `data/expression.csv` | Simulated differential expression results (generated by init.bl) |
| `data/gene_info.csv` | Gene annotations (generated by init.bl) |
| `scripts/analysis.bl` | Complete table analysis pipeline in BioLang |
| `scripts/analysis.py` | Python equivalent (pandas) |
| `scripts/analysis.R` | R equivalent (dplyr, tidyr) |
| `python/requirements.txt` | Python dependencies |
| `r/install.R` | R package installation |
| `compare.md` | Line-of-code comparison across languages |
| `expected/output.txt` | Expected console output |

## Quick Start

```bash
# Step 1: Generate sample data
cd days/day-10
bl run init.bl

# Step 2: Run the BioLang analysis
bl run scripts/analysis.bl

# Run the Python version
pip install -r python/requirements.txt
python scripts/analysis.py

# Run the R version
Rscript r/install.R
Rscript scripts/analysis.R
```
