# Day 8: Processing Large Files

| | |
|---|---|
| **Difficulty** | Intermediate |
| **Biology knowledge** | Light (file formats from Day 7) |
| **Coding knowledge** | Intermediate (streams, lazy evaluation, iterators) |
| **Time** | ~2 hours |
| **Prerequisites** | Days 1-7 completed, BioLang installed (see Appendix A) |
| **Data needed** | Generated by `init.bl` (synthetic FASTQ, FASTA, VCF, BED) |

## What You'll Learn

- The difference between eager (load-all) and streaming (one-at-a-time) file processing
- How to use streaming readers: `fastq()`, `fasta()`, `vcf()`, `bed()`, `bam()`, `gff()`
- Lazy pipeline construction with `filter()`, `map()`, `take()`, `drop()`, `tee()`
- Terminal operations: `count()`, `collect()`, `reduce()`, `first()`, `last()`, `frequencies()`
- Chunked processing with `stream_chunks()`
- Constant-memory patterns for files of any size

## Files

| File | Description |
|------|-------------|
| `init.bl` | Generates synthetic FASTQ (~500 reads), FASTA, VCF, BED files |
| `scripts/analysis.bl` | Complete streaming pipeline in BioLang |
| `scripts/analysis.py` | Python equivalent (generators, itertools) |
| `scripts/analysis.R` | R equivalent (connection-based reading, chunked) |
| `python/requirements.txt` | Python dependencies |
| `r/install.R` | R package installation |
| `compare.md` | Line-of-code comparison across languages |
| `expected/output.txt` | Expected console output |

## Quick Start

```bash
# Step 1: Generate sample data
bl run init.bl

# Step 2: Run the BioLang analysis
bl run scripts/analysis.bl

# Run the Python version
pip install -r python/requirements.txt
python scripts/analysis.py

# Run the R version
Rscript r/install.R
Rscript scripts/analysis.R
```
