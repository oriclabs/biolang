| | |

## What You'll Learn

- How to compute descriptive statistics and summarize data
- What p-values mean and how to interpret them correctly
- How to compare groups with t-tests, Wilcoxon, and ANOVA
- How to measure correlation and fit linear regression models
- Why multiple testing correction is critical in genomics
- How to test categorical associations with chi-square and Fisher's exact test
