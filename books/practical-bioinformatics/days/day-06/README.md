# Day 6: Reading Sequencing Data

| | |
|---|---|
| **Difficulty** | Intermediate |
| **Biology knowledge** | Intermediate (FASTQ format, Phred scores, sequencing QC) |
| **Coding knowledge** | Intermediate (pipes, filtering, streaming) |
| **Time** | ~2.5 hours |
| **Prerequisites** | Days 1-5 completed, BioLang installed (see Appendix A) |
| **Data needed** | Generated by `init.bl` (synthetic FASTQ with 100 reads) |

## What You'll Learn

- FASTQ file format: the four-line record structure
- Phred quality scores: what they mean and how to decode them
- Reading FASTQ files: eager loading vs streaming
- Computing quality statistics across reads
- Filtering reads by length, quality, and GC content
- Trimming low-quality bases from read ends
- Detecting and removing adapter contamination
- K-mer analysis for quality assessment
- Writing clean reads to a new FASTQ file
- Building a complete QC pipeline

## Files

| File | Description |
|------|-------------|
| `init.bl` | Generates synthetic FASTQ sample data for exercises |
| `data/manifest.json` | Describes sample data files |
| `scripts/analysis.bl` | Clean BioLang QC pipeline script |
| `scripts/analysis.py` | Python equivalent (Biopython, numpy) |
| `scripts/analysis.R` | R equivalent (ShortRead, Biostrings) |
| `python/requirements.txt` | Python dependencies |
| `r/install.R` | R package installation |
| `compare.md` | Line-of-code comparison across languages |
| `expected/output.txt` | Expected console output |

## Quick Start

```bash
# Step 1: Generate sample data
bl run init.bl

# Step 2: Run the BioLang analysis
bl run scripts/analysis.bl

# Run the Python version
pip install -r python/requirements.txt
python scripts/analysis.py

# Run the R version
Rscript r/install.R
Rscript scripts/analysis.R
```
