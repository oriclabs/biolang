## What You'll Learn

- How to run Over-Representation Analysis (ORA) and Gene Set Enrichment Analysis (GSEA)
- How to read GMT files and query GO, KEGG, Reactome, and STRING databases
- How to build interaction networks from gene lists
- How to run a complete enrichment pipeline from DE results to biological interpretation

## Files

| File | Description |
|------|-------------|

## Quick Start

```bash
cd days/day-16
bl init.bl
bl scripts/enrichment.bl
```

## Data Files Generated

- `data/hallmark.gmt` --- 8 Hallmark-style gene sets in GMT format (15 genes each)
- `data/de_results.csv` --- 50 genes with log2fc, baseMean, pvalue, padj columns
- `data/ranked_genes.csv` --- 100 genes ranked by log2 fold change (gene, score columns)
