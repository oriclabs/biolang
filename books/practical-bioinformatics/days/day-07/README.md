# Day 7: Bioinformatics File Formats

| | |
|---|---|
| **Difficulty** | Intermediate |
| **Biology knowledge** | Intermediate (genomic file formats, coordinate systems) |
| **Coding knowledge** | Intermediate (pipes, filtering, format conversion) |
| **Time** | ~2.5 hours |
| **Prerequisites** | Days 1-6 completed, BioLang installed (see Appendix A) |
| **Data needed** | Generated by `init.bl` (synthetic FASTA, VCF, BED, GFF, BAM) |

## What You'll Learn

- The five core bioinformatics file formats: FASTA, VCF, BED, GFF/GTF, SAM/BAM
- When to use each format and what information it stores
- Coordinate system differences (0-based half-open vs 1-based inclusive)
- Reading files with eager and streaming readers
- Converting between formats with correct coordinate translation
- Writing output in multiple formats

## Files

| File | Description |
|------|-------------|
| `init.bl` | Generates synthetic sample data for all five formats |
| `data/manifest.json` | Describes sample data files |
| `scripts/analysis.bl` | Complete BioLang multi-format analysis |
| `scripts/analysis.py` | Python equivalent (biopython, pysam, pandas) |
| `scripts/analysis.R` | R equivalent (Biostrings, VariantAnnotation, GenomicRanges, rtracklayer) |
| `python/requirements.txt` | Python dependencies |
| `r/install.R` | R package installation |
| `compare.md` | Line-of-code comparison across languages |
| `expected/output.txt` | Expected console output |

## Quick Start

```bash
# Step 1: Generate sample data
bl run init.bl

# Step 2: Run the BioLang analysis
bl run scripts/analysis.bl

# Run the Python version
pip install -r python/requirements.txt
python scripts/analysis.py

# Run the R version
Rscript r/install.R
Rscript scripts/analysis.R
```
