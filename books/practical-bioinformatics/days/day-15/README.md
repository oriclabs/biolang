## What You'll Learn

- How to choose the right plot for your data type
- How to create scatter plots, histograms, bar charts, and boxplots
- How to use bioinformatics-specific plots: volcano, MA, Manhattan, heatmap, genome track
- How to export SVG figures for publication
- Design principles for clear scientific figures

## Quick Start

```bash
cd days/day-15
bl init.bl
bl scripts/visualization.bl
```

## Data Files Generated

- `data/de_results.csv` --- 50 genes with log2fc, baseMean, pvalue, padj columns
- `data/gwas_results.csv` --- 50 variants across chromosomes with p-values
- `data/sample.fastq` --- 10 short FASTQ reads for quality visualization
