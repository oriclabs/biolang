# Day 19: Biological Data Visualization

| | |
|---|---|
| **Difficulty** | Intermediate |
| **Biology knowledge** | Intermediate (GWAS, expression, survival analysis, genomic structure) |
| **Coding knowledge** | Intermediate (tables, records, pipes, sets) |
| **Time** | ~3 hours |
| **Prerequisites** | Days 1-18 completed, BioLang installed (see Appendix A) |
| **Data needed** | Generated by `init.bl` |

## What You'll Learn

- How to create 21 specialized biological visualization types in BioLang
- Manhattan and QQ plots for GWAS results
- Violin, density, PCA, and clustered heatmap plots for expression data
- Kaplan-Meier, ROC, and forest plots for clinical data
- Ideograms, circos, and lollipop plots for genomic structure
- Sequence logos, phylogenetic trees, Venn diagrams, oncoprints, and more
- How to export publication-quality SVG figures

## Files

| File | Description |
|------|-------------|
| `init.bl` | Generates data files (GWAS CSV, expression matrix CSV) |
| `scripts/bio_viz.bl` | Complete BioLang bio visualization script |
| `scripts/bio_viz.py` | Python equivalent (matplotlib, seaborn, lifelines) |
| `scripts/bio_viz.R` | R equivalent (ggplot2, survival, pheatmap) |
| `python/requirements.txt` | Python dependencies |
| `r/install.R` | R package installation |
| `compare.md` | Language comparison (BioLang vs Python vs R) |
| `expected/output.txt` | Expected console output |

## Quick Start

```bash
# Generate data files
bl run init.bl

# Run the BioLang version
bl run scripts/bio_viz.bl

# Run the Python version
pip install -r python/requirements.txt
python scripts/bio_viz.py

# Run the R version
Rscript r/install.R
Rscript scripts/bio_viz.R
```
